这里提一个 盒型注释 Box Comment 的概念，形如：

```c
/* 
  ┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
  │ 有关c程序的构建规则，其实就是编排顺序，                                                                                            │
  │ 这里有对函数的编排，建议在每个函数定义前放盒型注释，                                                                                      │
  │ 可以给出函数名、函数目的、讨论每个参数的意义、返回值，还有函数会产生的副作用                                                                          │
  └─────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
 */
```
