预处理 preprocessor gcc- E 是把 .c 源代码处理成 .i

编译 compilation gcc -S 处理成 .s 特定的汇编代码

汇编 assemble gcc -c 把 .s 文件处理 .o

.o 是一个二进制文件

.out 是一个可执行文件

从 .o 到 .out 的过程就是一个链接 linking
