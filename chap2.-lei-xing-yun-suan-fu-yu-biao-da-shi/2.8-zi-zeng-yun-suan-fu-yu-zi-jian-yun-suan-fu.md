# 2.8 自增运算符与自减运算符

**Increment and Decrement Operators（递增和递减操作符）**:

* `++` : Increment（递增）
* `--` : Decrement（递减）

前缀使用与后缀使用，有一些差别。

如下是在字符串 s 中删除字符 c

{% code title="squeeze-function.c" lineNumbers="true" %}
```c
void squeeze(char s[], int c)
{
    int i,j;
    for (i=j=0; s[i] != '\0'; i++) 
    {
        if (s[j] != c) 
        {
            s[j++] = s[i];
        }
        s[j] = '\0\;
    }
}
```
{% endcode %}

先拷贝再递增，如下两个写法等价。

```c
if (s[j] != c) 
{
    s[j++] = s[i];
}
```

```c
if (s[j] != c) 
{
    s[j] = s[i];
    j++;
}
```
